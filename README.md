Dotfiles
========

Setup
-----

```
  git clone https://github.com/smith11235/dotfiles.git
  cd dotfiles
  rake deploy_dotfiles
```

Tmux
----

Use it.  Love it.

Ruby
----

For any ruby/rails project, from your root directory make and use a project specific rvm/ruby install.

### First Time Install

```bash
  cd my_project
  build_rvm
```

### On Each Login

```bash
  cd my_project
  load_rvm
```
